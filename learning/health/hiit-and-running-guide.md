# HIITとランニングの目的別ガイド

HIITを続けることでも、ランニングを長時間続ける能力はある程度向上する。HIITによって最大酸素摂取量（VO2max）や高い強度への耐性が高まり、同じランニングペースを以前より低い負担で維持しやすくなるためである。

ただし、HIITだけで長時間走に必要な能力をすべて身につけることは難しい。長く走るには、心肺機能以外にも、脚の筋肉・腱・関節の耐久性、ランニングフォームの効率、ペース配分、エネルギー補給などへの適応が必要になる。これらは、実際に余裕のあるペースで走る時間を徐々に延ばすことで身につきやすい。

したがって、次のように考えると分かりやすい。

> HIITは長く走るための「エンジン」を強くする。一定ペースのランニングは、そのエンジンを使って長時間走るための「車体と運用」を鍛える。

## 用語の違い

HIITとランニングは、本来は対立する分類ではない。

- HIIT（高強度インターバルトレーニング）は、高強度運動と低強度運動または休息を交互に行う運動方法である。
- ランニングは運動種目であり、一定ペースでも、インターバル形式でも実施できる。

そのため、このノートでは「HIIT」と「余裕のある一定ペースのランニング」を比較する。

## 目的別の比較

| 目的・特徴 | HIIT | 一定ペースのランニング |
| --- | --- | --- |
| 心肺機能 | 短時間で強い刺激を与えやすい | 継続することで十分に向上する |
| 長時間走る能力 | 心肺面から間接的に向上する | 筋・腱・フォームを含めて直接的に向上する |
| 高い速度への耐性 | 向上させやすい | ゆっくり走るだけでは向上しにくい |
| 1分当たりの消費エネルギー | 多くなりやすい | HIITより少ないことが多い |
| 1回の総消費エネルギー | 短時間では少なくなる場合がある | 長く続けやすいため確保しやすい |
| 疲労と回復 | 身体・神経系への負担が大きい | 余裕のあるペースなら回復しやすい |
| 実施頻度 | 通常は週1～2回で十分 | 負荷を管理すれば週に複数回行いやすい |
| 主な用途 | 時間効率、VO2max、スピード | 健康維持、持久力、走行距離、気分転換 |

HIITは、従来の一定強度の有酸素運動よりVO2maxを効率よく改善する傾向がある。一方で、一定ペースの有酸素運動にも明確な心肺機能向上効果があり、HIITだけが有効なわけではない。

## HIITで長時間走が楽になる理由

### 最大能力が高まり、同じペースの負担が下がる

HIITによって最大心肺能力が高まると、以前と同じ速度が自分の最大能力に占める割合が小さくなる。

たとえば同じ速度で走るために必要な能力が「40」の場合、最大能力が50の人には限界に近いが、最大能力が70の人には余裕がある。そのため、息切れしにくくなり、同じ速度を維持しやすくなる。

### 高い強度から回復する能力が高まる

速く走る区間と回復区間を繰り返すことで、心拍数が高い状態への耐性や、速度を落としたときの回復能力が向上する。坂道や一時的なペースアップにも対応しやすくなる。

## HIITだけでは不足しやすい能力

### 脚の組織の耐久性

長時間走では、同じ動作を何千回も繰り返す。心肺能力に余裕があっても、ふくらはぎ、膝、足首、腱などが負荷に慣れていなければ、痛みやフォームの崩れによって走れなくなる。

### ランニング効率

同じ速度を少ないエネルギーで維持する能力をランニングエコノミーという。HIITでも刺激できるが、余裕のあるペースで実際に走る練習も重要になる。

### エネルギー管理とペース配分

走る時間が長くなるほど、糖質の利用、体温、水分、ペース配分の影響が大きくなる。これらへの対応は、短いHIITだけでは経験しにくい。

## 消費カロリーと体脂肪

HIITは1分当たりの消費エネルギーが多く、運動後の酸素消費も一定ペース走より増えやすい。ただし、HIITは継続時間が短いため、1回全体では長めのランニングより総消費量が少ない場合がある。

体脂肪の減少は、運動方法の名前ではなく、長期的な消費エネルギー、摂取エネルギー、継続性によって大きく左右される。消費エネルギーをそろえた比較では、HIITが一定強度の有酸素運動より体脂肪減少で明確に優れるとは限らない。

## 目的に応じた選び方

| 主な目的 | 選び方 |
| --- | --- |
| 健康維持 | 継続しやすい一定ペース走を基本にする |
| 短時間で心肺機能を高める | HIITを週1～2回取り入れる |
| 30分、60分と長く走れるようになる | 余裕のあるランニング時間を徐々に延ばす |
| 走る速度を上げる | HIITと一定ペース走を組み合わせる |
| 消費エネルギーを増やす | 疲労を残さず長く続けられる運動を選ぶ |
| 体脂肪を減らす | 運動の継続に加えて食事全体も調整する |
| 対人・感覚刺激で強く疲れた日の回復 | HIITではなく軽いランニング、ウォーキングまたは休養を選ぶ |

## 3kmを20分程度走れる場合の週間例

健康維持と心肺機能の向上を両立するなら、すべてのランニングをHIITにする必要はない。

| 日 | 内容 |
| --- | --- |
| 1日目 | 3kmを会話できる程度の余裕のあるペースで走る |
| 2日目 | 休養またはウォーキング |
| 3日目 | 3kmを余裕のあるペースで走る |
| 4日目 | 休養または軽い筋力トレーニング |
| 5日目 | HIITまたは通常の3km走 |
| 6～7日目 | 疲労に応じて休養、ウォーキング、軽いランニングを選ぶ |

HIITは、次のような内容から始められる。

1. 5分間、歩くかゆっくり走って体を温める。
2. 30秒間速く走り、60～90秒間ゆっくり走る動作を4～6回繰り返す。
3. 5分間ゆっくり走るか歩いて終了する。

速い区間は全力疾走ではなく、自覚的なきつさを10段階で8程度にする。長時間走れるようになることが目的なら、HIITとは別の日に、余裕のあるランニングを5分程度ずつ延ばす。翌日まで強い疲労や痛みが残る場合は延長しない。

## 継続するときの注意

- HIITを連日行わない。
- 胸の痛み、失神しそうな感覚、異常な息苦しさがあれば中止する。
- 関節や腱の痛みが続く場合は、距離や強度を下げる。
- 強い対人疲労や睡眠不足がある日は、HIITを休養や軽い運動に変更する。
- 心臓や呼吸器の持病がある場合は、高強度運動を始める前に医療従事者へ相談する。

HIITは時間効率のよい方法だが、健康効果は最も激しい運動を選ぶことで決まるわけではない。目的に合った負荷を、回復可能な範囲で継続することが重要である。
