# Question logs

このディレクトリには、このプロジェクト内でユーザーが行った質問のうち、後から参照しそうな技術知識、またはユーザーが明示的に残すよう依頼した内容を日付別に保存します。一時的な確認、進捗確認、雑談、運用上のやり取りは記録しません。

各ログは `YYYY-MM-DD.md` という名前で、質問と回答を短く要約して次の表へ追記します。

```md
# YYYY-MM-DD 質問ログ

## QA一覧

| 時刻 | 質問要約 | 回答要約 |
| --- | --- | --- |
| HH:mm | 質問の要点 | 回答または対応結果の要点 |
```

一読して QA の内容が分かる粒度を保ち、長い経緯や会話表現は省きます。セル内の `|` は `\|` にエスケープします。

認証情報などの秘密情報は `[REDACTED]` に置き換えます。
